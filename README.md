# 🏓 Pong Game

A classic two-player Pong game built with Python's `turtle` module, featuring multiple game modes.

---

## 🚀 How to Run

1. Make sure you have Python installed.
2. Run the game:
   ```bash
   python main.py
   ```
3. Select your game mode when prompted.

---

## 🎮 Game Modes

| Mode | Description |
|------|-------------|
| **1 - First to 5** | First player to reach 5 points wins! |
| **2 - AI Opponent** | Play against the computer. Let's see if you can beat it! |
| **3 - Infinite / Casual** | No win condition — just play for fun. |

---

## 🕹️ Controls

| Player | Move Up | Move Down |
|--------|---------|-----------|
| Left Player | `W` | `S` |
| Right Player | `↑` | `↓` |

> In **AI mode**, the computer controls the right paddle.
